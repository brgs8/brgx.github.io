# Roteiro de Narração para NotebookLM

## Instrução geral
Usar tom seguro, objetivo e executivo.

## Direção de voz
- clara
- simples
- firme
- sem exagero
- sem termos confusos

## Estrutura recomendada para cada vídeo
1. problema ou oportunidade
2. conceito
3. ganho de negócio
4. fecho executivo

## Frase-base da série
`Menos improviso. Mais método. Mais resultado.`

## Ritmo recomendado
- primeira frase: chamar atenção com contexto de negócio
- segunda frase: explicar a ideia
- terceira frase: traduzir ganho operacional
- quarta frase: conectar com diretoria
- quinta frase: fechar com decisão

## Tom de linguagem
Falar como quem apresenta uma proposta de gestão, e não como quem faz uma campanha de marketing.

## Observação
Cada vídeo deve parecer uma decisão de gestão, não uma campanha promocional.

## Orientação para o NotebookLM
Se for gerar voz, manter:
- frases curtas
- pausas naturais
- vocabulário simples
- leitura limpa
- sem ênfase exagerada
