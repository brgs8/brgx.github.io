# Design do Web App

## Conceito visual
Relacional clean.

## Justificativa
O app precisa parecer leve, operacional e confiável, sem excesso de decoração.

## Paleta sugerida
- fundo escuro azulado
- superfícies translúcidas
- destaque em ciano e verde suave
- alertas em âmbar e vermelho suave

## Estilo dos componentes
- cartões com borda leve
- campos agrupados por etapa
- botões claros e objetivos
- ícones discretos
- estados visuais por status

## Estrutura das telas
1. Login
2. Página inicial
3. Novo registro
4. Consulta de registros
5. Detalhe do registro
6. Edição controlada
7. Dashboard
8. Administração
9. Configurações
10. Logs

## Hierarquia visual
1. tarefa do dia
2. registros pendentes
3. indicador principal
4. tabela de acompanhamento
5. administração e ajustes

## Responsividade
- desktop: menu lateral
- mobile: menu superior colapsável
- formulário em blocos verticais
- tabelas com visual adaptado para leitura rápida

## Como facilita o uso
- reduz cliques
- evita busca manual em várias planilhas
- mostra só o que importa para cada perfil
- dá leitura rápida para liderança
