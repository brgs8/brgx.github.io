# Guia de Implantação

1. criar a planilha base
2. criar as abas
3. inserir cabeçalhos
4. abrir Apps Script
5. criar `Code.gs`, `Index.html`, `Style.html`, `Script.html`
6. colar o código em cada arquivo
7. configurar e-mails e perfis
8. publicar como Web App
9. testar com usuários diferentes
10. validar permissões
11. corrigir erros comuns
12. manter backup periódico
