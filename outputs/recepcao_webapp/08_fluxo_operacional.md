# Fluxo Operacional

1. Usuário entra com e-mail Google.
2. Sistema identifica perfil.
3. Sistema carrega unidade e listas dinâmicas.
4. Usuário acessa painel inicial.
5. Usuário cria novo registro ou consulta registros existentes.
6. Sistema valida campos obrigatórios.
7. Sistema grava no Sheets.
8. Sistema escreve log.
9. Dashboard atualiza indicadores.
10. Perfis superiores acompanham unidade ou rede inteira.
