# Scripts de WhatsApp

## 1. Primeira resposta ao contato
```text
Oi, tudo bem?
Aqui é do Sala D’Oro em Alphaville.
Vi que você se interessou pelos nossos serviços exclusivos.
Vai ser um prazer te atender.
Qual serviço você está buscando hoje?
```

## 2. Qualificação rápida
```text
Perfeito.
Você prefere agendar para essa semana ou para a próxima?
```

## 3. Mensagem de diagnóstico
```text
Oi, tudo bem?
Para eu entender melhor o que você precisa, me responde rapidinho:

1. Qual serviço você quer estruturar ou vender melhor?
2. Hoje vocês já usam algum CRM ou controle?
3. Quem cuida da recepção e dos retornos?
4. O WhatsApp já é o canal principal de atendimento?
```

## 4. Envio de proposta
```text
Perfeito, já organizei sua necessidade.
Vou te enviar a proposta com a estrutura indicada para esse cenário.
Se fizer sentido, a gente já avança para a implantação.
```

## 5. Retomada de negociação
```text
Oi, passando para retomar nosso contato.
A proposta que te enviei continua atual para o que vocês precisam?
Se quiser, eu posso te mostrar o melhor caminho para começar de forma simples.
```

## 6. Agendamento de reunião
```text
Posso te sugerir um encontro rápido para alinharmos o diagnóstico e os próximos passos?
Se preferir, eu já te mando duas opções de horário.
```

## 7. Boas-vindas ao cliente
```text
Bem-vinda ao projeto.
Agora eu vou organizar o fluxo inicial, os materiais de apoio e a rotina de acompanhamento para deixar tudo pronto de forma simples e prática.
```
