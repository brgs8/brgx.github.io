# Relacionamento

SPA leve para a equipe de acolhimento/relacionamento.

## Como usar

1. Abra `relacionamento-app/index.html` no navegador.
2. Entre com a senha padrão de teste: `1234`.
3. Registre as execuções do dia e acompanhe o painel.

## O que já faz

- Login simples por senha
- Rotina por dia da semana
- Formulário de execução com campos variáveis
- Lista de evidências e observações
- Dashboard local com progresso

## Observação

Os dados ficam salvos localmente no navegador nesta primeira versão.
